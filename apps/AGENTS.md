# Application Services

Каталог содержит прикладные сервисы стенда: wallet API, clickstream API, transaction emulator и wallet UI.

## Общие Требования

- Сервисы должны запускаться из Docker Compose без ручной подготовки окружения.
- Не меняй публичные HTTP-контракты без синхронного обновления UI, тестов, документации и ETL-зависимостей.
- JSON-поля должны оставаться совместимыми с текущими Spark jobs и Superset/StarRocks витринами.
- Для Python-сервисов используй FastAPI/Pydantic-паттерны, уже принятые в проекте.
- Для frontend используй существующий React/Vite stack и текущий визуальный стиль.
- Тесты сервисов должны оставаться быстрыми и не требовать внешнего PostgreSQL/Kafka, если это unit-level проверка.
