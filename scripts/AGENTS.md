# Scripts

Локальные вспомогательные скрипты для smoke checks и demo automation.

## Требования

- Скрипты должны быть безопасными для повторного запуска.
- Используй POSIX-compatible shell, если нет явной причины требовать bash.
- Не добавляй destructive operations (`rm`, volume prune, database drop) без отдельной явной команды и предупреждения.
- Скрипты должны работать из корня репозитория.
- Если скрипт зависит от запущенного Compose-стенда, явно проверяй или документируй это.
